# Topological Time-Series Analysis — Replication Code

Replication repository for the master's thesis
*"Topological Data Analysis for Time-Series Regime Detection and Forecasting"*
by Stanisław M. S. Halkiewicz, AGH University of Science and Technology, 2026.

## Setup

```bash
git clone https://github.com/profsms/AGH_TDA-thesis-replication.git
cd thesis-replication

julia --project=. -e "using Pkg; Pkg.instantiate()"
```

---

## Data

None of the raw data is committed to this repository. Obtain each dataset as follows.

### Financial series (scripts 08–)
```bash
julia --project=. 08a_download_all.jl
```
Downloads 15 daily price series from Yahoo Finance into `data/`.

### M4 competition (scripts 09–)
```bash
julia --project=. 09a_download_m4.jl
```
Downloads the official M4 daily train/test sets from the Mcompetitions GitHub into `data/m4/`.

### Sunspots (scripts 10–)
```bash
julia --project=. 10a_download_sunspots.jl
```
Downloads daily total sunspot numbers from SILSO (Royal Observatory of Belgium) into `data/`.

### Victorian electricity (scripts 11–)
```bash
julia --project=. 11a_download_elec.jl
```
Downloads half-hourly demand data from the tsibbledata GitHub into `data/`.

### NASA IMS bearing dataset (scripts 02–05)
Download the IMS bearing dataset manually from the
[NASA Prognostics Data Repository](https://www.nasa.gov/intelligent-systems-division/discovery-and-systems-health/pcoe/pcoe-data-set-repository/)
and extract it so the directory tree looks like:

```
IMS/
  1st_test/    (2 156 files)
  2nd_test/    (984 files)
  3rd_test/    (6 324 files)
  Readme Document for IMS Bearing Data.pdf
```

Then preprocess it (downsampled, ~160 MB total):
```bash
julia --project=. preprocess_nasa.jl
```
This writes `data/nasa_set1.jls`, `data/nasa_set2.jls`, `data/nasa_set3.jls`.

## Running the analysis

Scripts are numbered in execution order. Each script saves checkpoints under
`checkpoints/` and figures under `plots/`. Final numeric results are written to
`results/`.

### Part I — Synthetic changepoint detection

```bash
julia --threads=auto --project=. 01_synthetic.jl
julia --threads=auto --project=. 01b_synthetic_phasespace.jl
```

Replicates El-Yaagoubi et al. (2025) and benchmarks the novel phase-space pipeline
across three signal families and three bandwidths.

### Part II — NASA IMS bearing dataset

```bash
# Requires data/nasa_set*.jls from the preprocess step above
julia --threads=auto --project=. 02_nasa.jl
julia --project=. 03_nasa_detection.jl
julia --project=. 04_nasa_m5_plots.jl
julia --project=. 05_nasa_crocker.jl
```

Regime-detection experiments on roller-element bearing run-to-failure signals.

### Part III — Signal characterisation & period estimation

```bash
julia --project=. 06_signal_characterisation.jl
julia --project=. 07_period_estimation.jl
```

§6.1 signal-property plots and Monte Carlo comparison of topological vs. FFT period
estimators.

### Part IV — Forecasting benchmark

```bash
# Financial (requires data/ from 08a)
julia --project=. 08b_forecasting.jl

# M4 competition (requires data/m4/ from 09a)
julia --project=. 09b_m4_forecasting.jl

# Sunspots (requires data/sunspots_daily.csv from 10a)
julia --project=. 10b_sunspot_forecasting.jl

# Victorian electricity (requires data/vic_elec.csv from 11a)
julia --project=. 11b_elec_forecasting.jl
```

XGBoost walk-forward benchmark: baseline lag features vs. baseline + topological
features at horizons h = 1 and h = 5.
